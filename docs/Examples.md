# Example Plugins

The repository includes example plugins you can copy into your local plugin directory.

## Included Examples

- `hello_plugin`
- `timer_plugin`
- `clock_usage_plugin`
- `api_tour_plugin`
- `stateful_counter_plugin`
- `channel_toolbox_plugin`

## What They Show

- `hello_plugin`: the smallest possible command registration example
- `timer_plugin`: delayed local messages with `c2.later`
- `clock_usage_plugin`: time formatting, persistent runtime, and settings-aware 12h/24h display
- `api_tour_plugin`: account lookup, channel lookup, sending messages, clearing messages, browser open, delayed callbacks, and completion callbacks
- `stateful_counter_plugin`: persistent plugin state and completion hooks
- `channel_toolbox_plugin`: channel lookup, sending messages, local notes, and clearing message buffers
